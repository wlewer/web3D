### 2. 状态机测试

#### ✅ 状态转换验证
打开浏览器控制台，执行以下操作并观察日志：

1. **页面加载**
   ```
   预期：UNINITIALIZED → INITIALIZING → READY → LOADING → LOADED
   ```

2. **切换模型**
   ```
   预期：LOADED → LOADING → LOADED
   ```

3. **快速连续切换**
   - 快速点击左右箭头5次
   - 观察是否有重复加载或竞态条件
   
   **预期**：每次切换都按顺序执行，无并发加载

## 🔍 调试技巧

在浏览器控制台中执行：
```javascript
// 获取UniversalGaussianCardV2组件实例
const card = document.querySelector('.universal-card-v2');
// 通过React DevTools查看组件状态
```

### 2. 模拟慢速网络

1. 打开Chrome DevTools
2. 切换到Network标签
3. 选择"Slow 3G"预设
4. 刷新页面，观察加载行为

### 3. 强制触发错误

```javascript
// 临时修改modelUrl为无效值
// 在HomePage.tsx中：
modelUrl="https://invalid-url.com/model.spz"
```
