## 🚀 使用方法

### 访问方式
1. 启动开发服务器：`npm run dev`
2. 打开浏览器：`http://localhost:5173`
3. 点击底部"📚 3D翻页书"按钮

### 操作说明
- **翻页**：点击"上一页"/"下一页"按钮
- **键盘**：← 左箭头（上一页），→ 右箭头或空格（下一页）
- **视角**：鼠标拖拽旋转，滚轮缩放

## 🔧 技术实现

### 物理引擎
- **翻页角度计算**：`calculateFlipAngle()` - 0到π弧度
- **纸张弯曲**：`calculatePaperBend()` - 正弦函数模拟
- **缓动函数**：`easeInOutCubic()` - 三次方缓入缓出

### React Three Fiber
- **Canvas**：3D渲染容器
- **useFrame**：动画帧更新
- **OrbitControls**：相机控制
- **Environment**：环境光照

### 状态管理
- **useBookFlip Hook**：统一管理翻页状态
- **currentPage**：当前页码
- **isFlipping**：翻页动画状态
- **flipProgress**：翻页进度 0-1

## 🎨 可扩展性

### 自定义配置
```typescript
<Book3D 
  totalPages={20}
  config={{
    width: 2.5,        // 书本宽度
    height: 3.5,       // 书本高度
    flipDuration: 1.2, // 翻页时长（秒）
  }}
/>
```

### 添加页面内容
修改 `Book3D.tsx` 中的 `pages` 数组：
```typescript
const pages = [
  {
    id: 'page-1',
    frontContent: <YourFrontComponent />,
    backContent: <YourBackComponent />,
  },
  // ...
];
```
