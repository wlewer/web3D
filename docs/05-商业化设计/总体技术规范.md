## 3. 统一编码规范

### 3.1 命名规范

| 场景 | 规范 | 示例 |
|------|------|------|
| 文件/目录 | kebab-case | `model-service.ts`, `user-profile/` |
| React 组件 | PascalCase | `ModelViewer.tsx` |
| Hooks | camelCase + use 前缀 | `useModelLoader.ts` |
| 类型/接口 | PascalCase + 语义后缀 | `ModelSchema`, `UserDTO` |
| 常量 | SCREAMING_SNAKE_CASE | `MAX_UPLOAD_SIZE_MB` |
| API 端点 | kebab-case + 复数 | `GET /api/v1/models` |
| 数据库表 | snake_case + 复数 | `template_slots`, `generated_models` |
| 数据库字段 | snake_case | `is_dynamic`, `created_at` |
| Python 模块 | snake_case | `model_service.py` |
| Python 类 | PascalCase | `ModelService` |

### 3.2 目录组织规范

前端 (`src/web-frontend/src/`):
```
src/
├── components/          # 原子/分子组件（纯展示）
├── features/            # 业务功能模块（按域组织）
│   ├── model-gallery/

├── stores/              # Zustand 状态定义
```

## 4. API 设计规范

### 4.1 RESTful 路径规则

```
/api/v1/{resource}           # 列表 / 创建
/api/v1/{resource}/{id}      # 详情 / 更新 / 删除
/api/v1/{resource}/{id}/action  # 自定义操作
```

### 4.2 统一响应格式

```json
{
  "code": 0,
  "message": "success",
  "data": { },
  "request_id": "uuid",
  "timestamp": 1715923200
}
```

### 4.3 错误码定义

| 错误码 | 含义 | HTTP 状态 |
|--------|------|-----------|
| 0 | 成功 | 200 |
| 1001 | 参数校验失败 | 400 |
| 1002 | 认证失败 | 401 |
| 1003 | 权限不足 | 403 |
| 1004 | 资源不存在 | 404 |
| 1005 | 请求过于频繁 | 429 |
| 2001 | 租户已禁用 | 403 |
| 2002 | 配额已耗尽 | 403 |
| 3001 | 模型处理失败 | 500 |
| 3002 | 存储服务异常 | 500 |
| 9999 | 系统内部错误 | 500 |

## 5. 数据库设计规范

### 5.1 命名规范

- 表名：复数形式，`snake_case`
- 字段：`snake_case`，禁止 SQL 保留字（如 `metadata`）
- 索引：`idx_{table}_{field}`
- 外键：`fk_{table}_{ref_table}`

### 5.2 必含字段

```sql
created_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
updated_at  TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
deleted_at  TIMESTAMP NULL DEFAULT NULL,  -- 软删除
created_by  INT NULL,
updated_by  INT NULL
```

### 5.3 迁移规范

- 使用 Alembic 管理迁移脚本
- 脚本命名：`{timestamp}_{description}.py`
- 禁止直接修改已执行的迁移文件
- 生产环境迁移必须先在 staging 验证

## 6. 前端组件规范

### 6.1 Props 接口定义

```typescript
// types/model-viewer.ts
export interface ModelViewerProps {
  modelUrl: string;
  fileType: 'glb' | 'spz' | 'ply';
  autoCenter?: boolean;
  labelCount?: number;
  onLoad?: () => void;
  onError?: (err: Error) => void;
}
```

### 6.2 状态管理

- 全局状态（跨模块）：Zustand store
- 模块级状态：React Context + useReducer
- 组件级状态：useState / useReducer

### 6.3 样式方案

- 管理后台：Ant Design 主题变量 + CSS Modules
- 官网/展示页：Tailwind CSS + 设计 Token
- 3D 画布：行内样式 + CSS 变量（控制提示、Stats 面板）
