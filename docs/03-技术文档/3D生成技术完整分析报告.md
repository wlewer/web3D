## 四、各技术详细说明

### 4.1 Hunyuan3D-2（腾讯混元）⭐⭐⭐⭐⭐

**最强3D生成引擎**

| 属性 | 内容 |
|:---|:---|
| GitHub | https://github.com/Tencent-Hunyuan/Hunyuan3D-2 |
| Stars | 13.5k |
| 开发方 | 腾讯混元 (Tencent Hunyuan) |
| 发布时间 | 2025年1月21日 |

**核心架构**：
```
输入图像/文本 ──► 几何生成阶段 ──► 纹理合成阶段
              Hunyuan3D-DiT      Hunyuan3D-Paint
                  裸网格            高分辨率纹理贴图
```

**模型版本**：

| 模型 | 参数量 | 显存需求 |
|:---|:---|:---|
| Hunyuan3D-DiT-v2.1 | 3.0B | 6GB（仅形状） |
| Hunyuan3D-Paint-v2.1 | 1.3B | 16GB（形状+纹理） |
| Hunyuan3D-2mini | 0.6B | 更低显存 |

**API使用**：
```python
from hy3dgen.shapegen import Hunyuan3DDiTFlowMatchingPipeline
from hy3dgen.texgen import Hunyuan3DPaintPipeline

# 形状生成
shape_pipeline = Hunyuan3DDiTFlowMatchingPipeline.from_pretrained('tencent/Hunyuan3D-2')
mesh = shape_pipeline(image='demo.png')[0]

# 纹理生成
tex_pipeline = Hunyuan3DPaintPipeline.from_pretrained('tencent/Hunyuan3D-2')
mesh = tex_pipeline(mesh, image='demo.png')

mesh.export('output.glb')
```

### 4.2 TRELLIS.2（微软）⭐⭐⭐⭐⭐

**最高质量PBR材质**

| 属性 | 内容 |
|:---|:---|
| GitHub | https://github.com/microsoft/TRELLIS.2 |
| Stars | 5.5k |
| 开发方 | Microsoft Research |
| 模型规模 | 40亿参数 (4B) |

**核心能力**：
- O-Voxel技术：支持开放表面（衣物、树叶）、非流形几何
- 完整PBR材质：Base Color、Roughness、Metallic、Opacity
- 多分辨率输出：512³（3秒）、1024³（17秒）、1536³（60秒）

**技术架构**：
```
输入图像 → 稀疏结构流模型 → 形状/材质生成 → O-Voxel表示 → 可渲染3D资产
              ↑
        [4B参数DiT扩散模型]
```

### 4.3 Spark 2.0（World Labs）⭐⭐⭐⭐⭐

**浏览器端3DGS渲染王者**

| 属性 | 内容 |
|:---|:---|
| GitHub | https://github.com/sparkjsdev/spark |
| Stars | 2.6k |
| 最新版本 | v2.0.0 (2026-04-14) |
| 技术栈 | TypeScript 58.5% + Rust 35.7% |

**三大核心技术**：

1. **LoD细节层级系统**：恒定splats预算，平滑过渡
2. **.RAD渐进式流式传输**：64K chunks，3 Worker并行
3. **虚拟内存页表**：16M GPU显存池，LRU置换

**性能表现**：

| 场景 | Splats数量 |
|:---|:---:|
| 波兰扫描 | **106M** |
| 洞穴 | 73M |
| Coit Tower | 40M |
| 霍比屯 | 24M |
| 手机流畅运行 | 4000万 |

**前端集成**：
```javascript
import * as THREE from "three";
import { SplatMesh } from "@sparkjsdev/spark";

const splatURL = "https://example.com/scene.spz";
const splat = new SplatMesh({ url: splatURL });
scene.add(splat);
```
