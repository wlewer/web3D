## 📝 使用说明

### 快速开始（3步）

```bash
# 1. 启动服务
start_demo.bat          # Windows
./start_demo.sh         # Mac/Linux

# 2. 访问页面
http://localhost:5173

# 3. 点击"⚡ AI生成"并上传图片
```

### 模式切换

```bash
# 编辑 backend/.env
HUNYUAN3D_MODE=mock     # Mock模式（默认）
HUNYUAN3D_MODE=local    # Local模式
HUNYUAN3D_MODE=cloud    # Cloud模式

# 重启后端服务
```
