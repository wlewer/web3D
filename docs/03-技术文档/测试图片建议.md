# 测试图片建议

## 📸 推荐图片类型

为了获得最佳的AI生成效果（Local/Cloud模式），建议使用以下类型的图片：

### ✅ 推荐的图片

1. **单一主体**
   - 一个清晰的物体（如椅子、花瓶、玩具等）
   - 背景简洁或纯色
   - 主体占据画面60%以上

2. **多角度可见**
   - 物体有明确的立体结构
   - 不是完全扁平的（如纸张）
   - 有明显的深度信息

3. **高质量清晰**
   - 分辨率至少 512x512
   - 光线充足，无明显阴影
   - 对焦清晰，无模糊

4. **常见物体**
   - 家具（椅子、桌子、柜子）
   - 日用品（杯子、瓶子、盒子）
   - 玩具模型
   - 动物（猫、狗等）

### ❌ 不推荐的图片

1. **复杂场景**
   - 多人合影
   - 风景照
   - 室内全景

2. **抽象内容**
   - 文字为主
   - 纯图案
   - 艺术画作

3. **低质量图片**
   - 分辨率过低（< 256x256）
   - 严重模糊
   - 过暗或过亮

4. **特殊角度**
   - 完全正面（无侧面信息）
   - 完全俯视/仰视
   - 遮挡严重

## 🎯 Mock模式测试

**Mock模式对图片没有要求！**

- 可以上传任意图片
- 仅用于验证技术流程
- 不会真正生成3D模型
- 5秒后返回模拟结果

**建议：**
- 使用项目中的示例图片：`src/imgs/UI.png`
- 或使用手机拍摄的任意清晰照片

## 📱 手机端拍摄建议

如果用手机拍摄测试图片：

### 拍摄技巧

1. **光线充足**
   - 白天自然光最佳
   - 避免强烈逆光
   - 可以使用台灯补光

2. **保持稳定**
   - 双手持机
   - 靠在桌面上
   - 使用三脚架（可选）

3. **距离适中**
   - 距离物体30-50cm
   - 确保物体完整在画面中
   - 不要离得太近或太远

4. **角度选择**
   - 45度角拍摄（能看到侧面）
   - 稍微俯视
   - 避免完全正面

### 示例场景

```
场景1：书桌上的水杯
  ✓ 光线：窗边自然光
  ✓ 角度：45度斜拍
  ✓ 背景：纯色桌面
  ✓ 距离：40cm

场景2：椅子
  ✓ 光线：室内灯光
  ✓ 角度：侧前方
  ✓ 背景：白色墙壁
  ✓ 距离：1米

场景3：玩具模型
  ✓ 光线：台灯补光
  ✓ 角度：略微俯视
  ✓ 背景：干净桌面
  ✓ 距离：30cm
```

## 🔍 图片检查清单

上传前检查：

- [ ] 图片清晰，无模糊
- [ ] 主体明确，容易识别
- [ ] 背景相对简洁
- [ ] 光线充足，无严重阴影
- [ ] 分辨率足够（> 512x512）
- [ ] 文件格式为 JPG 或 PNG

## 📊 不同模式的图片要求

| 模式 | 图片要求 | 用途 |
|------|---------|------|
| **Mock** | 无要求 | 技术演示、流程验证 |
| **Local** | 高质量、单一主体 | 真实AI生成测试 |
| **Cloud** | 高质量、单一主体 | 生产环境使用 |

## 💡 提示

1. **Mock模式**：任何图片都可以，重点是测试流程

3. **多次尝试**：如果第一次效果不好，换一张图片试试
4. **参考示例**：查看Hunyuan3D官方示例图片
