## 🎯 下一步行动

### 立即执行（今天）

1. ⏳ **安装 Hunyuan3D 依赖**
   ```bash
   cd src/hunyuan3d
   pip install -r requirements.txt
   ```

2. ⏳ **配置 HuggingFace Token**
   ```bash
   export HF_TOKEN=your_token
   ```

3. ⏳ **启动 API 服务器**
   ```bash
   python api_server.py --model_path tencent/Hunyuan3D-2mini --port 8081
   ```

4. ⏳ **测试生成接口**
   ```bash
   python test_api.py
   ```
