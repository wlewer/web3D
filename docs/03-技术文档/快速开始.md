# 🚀 快速开始 - Hunyuan3D AI生成服务

## ⚡ 5分钟快速演示

### 第1步：启动服务（30秒）

```bash
# Windows
start_demo.bat

# Mac/Linux
chmod +x start_demo.sh && ./start_demo.sh
```

### 第2步：访问页面（10秒）

打开浏览器：**http://localhost:5173**

### 第3步：测试生成（2分钟）

1. 点击导航栏 **"⚡ AI生成"**
2. 点击 **"上传图片生成 3D 模型"**
3. 选择任意图片（建议使用 `src/imgs/UI.png`）
4. 等待5秒，查看进度条
5. 生成完成后点击 **"下载 GLB 模型"**

### 第4步：手机端测试（2分钟）

```bash
# 查看电脑IP
ipconfig          # Windows
ifconfig          # Mac/Linux

# 手机浏览器访问
http://[你的IP]:5173
```

## 🔧 切换到真实生成

### Local模式（需要GPU）

```bash
# 1. 编辑配置文件
cd backend
cp .env.example .env
# 修改：HUNYUAN3D_MODE=local

# 2. 部署Hunyuan3D服务
git clone https://github.com/Tencent/Hunyuan3D-1.git
cd Hunyuan3D-1
pip install -r requirements.txt
python api_server.py --host 0.0.0.0 --port 8081

# 3. 重启后端
# Ctrl+C 停止，然后重新启动
```

### Cloud模式（需要API Key）

```bash
# 1. 申请API Key
# 访问：https://hy3d.dev

# 2. 编辑配置文件
cd backend
# 修改 .env：
# HUNYUAN3D_MODE=cloud
# HUNYUAN3D_CLOUD_API_KEY=your_key

# 3. 重启后端
```

## ❓ 常见问题

### Q: 启动失败怎么办？

```bash
# 检查Python版本
python --version  # 需要 3.9+

# 检查Node版本
node --version    # 需要 18+

# 手动启动后端
cd backend
python -m uvicorn app.main:app --reload

# 手动启动前端
cd src/web-frontend
npm run dev
```

### Q: 手机端无法访问？

1. 确认同一WiFi网络
2. 检查防火墙设置
3. 尝试使用ngrok内网穿透

### Q: 如何停止服务？

```bash
# Windows: 关闭命令行窗口
# Mac/Linux: 
kill $(lsof -t -i:8000)  # 停止后端
kill $(lsof -t -i:5173)  # 停止前端
```
