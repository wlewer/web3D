## 🚀 本地测试步骤

### 1. 启动后端服务

```bash
cd d:\HBuilderProjects\web3D\backend

# 激活虚拟环境
.venv\Scripts\activate

# 启动服务
python -m uvicorn app.main:app --reload --port 8000
```

访问 API 文档: http://localhost:8000/docs

---

### 2. 启动前端服务

```bash
cd d:\HBuilderProjects\web3D\src\web-frontend

# 安装依赖（如果还没装）
npm install

# 启动开发服务器
npm run dev
```

访问前端: http://localhost:5175

---

### 3. 测试用户系统

1. **注册账号**:
   - 访问 http://localhost:5175/auth
   - 填写用户名、邮箱、密码
   - 点击注册

2. **登录**:
   - 输入用户名和密码
   - 获取 JWT Token

3. **查看个人信息**:
   - 访问用户中心
   - 验证信息显示正确

---

### 4. 测试模型上传

1. **准备测试文件**:
   - 找一个小的 GLB 文件（< 10MB）
   - 或使用示例文件

2. **上传模型**:
   ```bash
   curl -X POST "http://localhost:8000/api/v1/models/upload" \
     -H "Authorization: Bearer YOUR_TOKEN" \
     -F "file=@test.glb" \
     -F "name=Test Model" \
     -F "category=product"
   ```

3. **验证上传**:
   - 检查 `backend/uploads/models/` 目录
   - 确认文件已保存
   - 检查数据库记录

4. **下载模型**:
   ```bash
   curl -X GET "http://localhost:8000/api/v1/models/MODEL_ID/download" \
     -H "Authorization: Bearer YOUR_TOKEN" \
     -o downloaded.glb
   ```

5. **对比文件**:
   ```bash
   # Windows PowerShell
   Get-FileHash test.glb
   Get-FileHash downloaded.glb
   ```

---

### 5. 测试模板管理

1. **创建模板**:
   ```bash
   curl -X POST "http://localhost:8000/api/v1/templates/" \
     -H "Authorization: Bearer YOUR_TOKEN" \
     -H "Content-Type: application/json" \
     -d '{
       "name": "Test Template",
       "category": "product",
       "spark_config": {
         "backgroundColor": "#1a1a2e",
         "cameraPosition": [0, 2, 5]
       }
     }'
   ```

2. **获取列表**:
   ```bash
   curl "http://localhost:8000/api/v1/templates/" \
     -H "Authorization: Bearer YOUR_TOKEN"
   ```

3. **更新模板**:
   ```bash
   curl -X PUT "http://localhost:8000/api/v1/templates/TEMPLATE_ID" \
     -H "Authorization: Bearer YOUR_TOKEN" \
     -H "Content-Type: application/json" \
     -d '{
       "description": "Updated description"
     }'
   ```

4. **创建版本**:
   ```bash
   curl -X POST "http://localhost:8000/api/v1/templates/TEMPLATE_ID/versions" \
     -H "Authorization: Bearer YOUR_TOKEN" \
     -H "Content-Type: application/json" \
     -d '{
       "version": "1.1.0",
       "spark_config": {...},
       "change_log": "Updated camera position"
     }'
   ```

### 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 401 Unauthorized | Token 过期或无效 | 重新登录获取新 Token |
| 403 Forbidden | 权限不足 | 使用管理员账号 |
| 404 Not Found | 资源不存在 | 检查 ID 是否正确 |
| 500 Internal Error | 服务器错误 | 查看后端日志 |
| CORS Error | 跨域问题 | 检查 CORS 配置 |
