# 快速测试指南 - Hunyuan3D AI生成服务

## 🎯 目标

在5分钟内完成本地与手机端的打通测试，验证"图生3D"完整流程。

---

## ✅ 前置条件

- Node.js 18+ 已安装
- Python 3.9+ 已安装
- 浏览器（Chrome/Edge/Safari）

---

## 🚀 快速开始（3步）

### 步骤1：启动服务

```bash
# Windows用户
start_demo.bat

# Mac/Linux用户
chmod +x start_demo.sh
./start_demo.sh
```

等待约10秒，看到以下提示表示启动成功：
```
========================================
  启动完成！
========================================
```

### 步骤2：访问页面

**电脑端：**
- 打开浏览器访问：http://localhost:5173
- 点击导航栏的 **"⚡ AI生成"**

**手机端：**
1. 查看电脑IP地址：
   ```bash
   # Windows
   ipconfig
   # 找到 "IPv4 地址"，例如：192.168.1.100
   
   # Mac/Linux
   ifconfig
   # 找到 "inet"，例如：192.168.1.100
   ```

2. 在手机浏览器访问：`http://[你的IP]:5173`
   - 例如：`http://192.168.1.100:5173`

3. 点击 **"⚡ AI生成"**

### 步骤3：上传测试

1. 点击 **"上传图片生成 3D 模型"** 按钮
2. 选择一张清晰的图片（建议主体明确）
3. 观察进度条变化
4. 等待5秒后显示"生成完成"
5. 点击 **"下载 GLB 模型"**

## 🔍 问题排查

### 问题1：前端无法访问

**症状：** 浏览器显示"无法连接"

**解决：**
```bash
# 检查前端是否启动
cd src/web-frontend
npm run dev

# 确认端口占用
netstat -ano | findstr :5173  # Windows
lsof -i :5173                  # Mac/Linux
```

### 问题2：后端启动失败

**症状：** 命令行报错

**解决：**
```bash
# 检查Python版本
python --version  # 需要 3.9+

# 安装依赖
cd backend
pip install -r requirements.txt

# 手动启动测试
python -m uvicorn app.main:app --reload
```

### 问题3：手机端无法访问

**症状：** 手机浏览器显示"无法连接"

**解决：**
1. 确认手机和电脑在同一WiFi
2. 检查防火墙设置：
   ```bash
   # Windows防火墙允许端口
   netsh advfirewall firewall add rule name="Web3D Frontend" dir=in action=allow protocol=TCP localport=5173
   netsh advfirewall firewall add rule name="Web3D Backend" dir=in action=allow protocol=TCP localport=8000
   ```
3. 尝试使用内网穿透工具（ngrok）

### 问题4：上传后无反应

**症状：** 点击上传后没有进度条

**解决：**
1. 打开浏览器开发者工具（F12）
2. 查看Console标签是否有错误
3. 查看Network标签，确认API请求是否发送
4. 检查后端日志：
   ```bash
   # Windows
   type logs\backend.log
   
   # Mac/Linux
   tail -f logs/backend.log
   ```
