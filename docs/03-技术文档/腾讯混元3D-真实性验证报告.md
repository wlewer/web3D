```python
async def _download_glb(self, download_url: str) -> bytes:
    """从URL下载GLB文件"""
    logger.info(f"[Hunyuan3D Cloud] 开始下载GLB: {download_url}")
    
    async with aiohttp.ClientSession() as session:
        async with session.get(download_url, timeout=self.timeout) as response:
            if response.status != 200:
                raise Exception(f"下载GLB失败: HTTP {response.status}")
            
            glb_data = await response.read()
            logger.info(f"[Hunyuan3D Cloud] GLB下载完成: {len(glb_data)} bytes")
            return glb_data
```

### 8️⃣ 完整的generate方法流程 - ✅ 正确

**代码检查**（第97-192行）：
```python
async def generate(self, image_path: str, output_path: str, ...) -> Dict[str, Any]:
    try:
        start_time = time.time()
        
        # 1. 读取图片并转为base64
        with open(image_path, 'rb') as f:
            image_data = f.read()
        image_base64 = base64.b64encode(image_data).decode('utf-8')
        
        # 2. 提交生成任务
        task_id = await self._submit_task(image_base64=image_base64, prompt=prompt)
        
        # 3. 轮询查询任务状态
        result_data = await self._poll_task_status(task_id)
        
        if not result_data:
            return {'success': False, 'error': '任务超时或失败'}
        
        # 4. 下载GLB文件
        download_url = result_data.get('ResultUrl')
        if not download_url:
            return {'success': False, 'error': '响应中没有GLB下载链接'}
        
        # 5. 下载GLB文件
        glb_data = await self._download_glb(download_url)
        
        # 6. 保存GLB文件
        output_path_obj = Path(output_path)
        output_path_obj.parent.mkdir(parents=True, exist_ok=True)
        output_path_obj.write_bytes(glb_data)
        
        return {
            'success': True,
            'output_path': str(output_path),
            'generation_time': elapsed,
            'task_id': task_id,
            'file_size': file_size
        }
        
    except Exception as e:
        logger.error(f"[Hunyuan3D Cloud] 生成失败: {e}", exc_info=True)
        return {'success': False, 'error': str(e)}
```

**流程验证**：
1. ✅ 读取图片 → Base64编码
2. ✅ 提交任务 → 获取JobId
3. ✅ 轮询状态 → 等待DONE
4. ✅ 解析结果 → 提取GLB URL
5. ✅ 下载文件 → 获取二进制数据
6. ✅ 保存文件 → 写入本地磁盘
7. ✅ 返回结果 → success=True + 文件路径
