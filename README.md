# 3DGS技术官网项目

## 项目简介

基于 **React + Three.js + Spark 2.0** 的 3DGS 技术官网，实现亿级点云流畅渲染和手机拍照建模功能。

---

## 技术栈

| 层级 | 技术选型 |
|:---|:---|
| **前端框架** | React 18 + Vite + TypeScript |
| **3D渲染** | Three.js + React Three Fiber + Spark 2.0 |
| **UI组件** | Ant Design 5 + Tailwind CSS |
| **后端服务** | Python + FastAPI |
| **数据库** | PostgreSQL + Redis |

## 快速开始

```bash
# 安装依赖
npm install

# 安装3D依赖
npm install three @react-three/fiber @react-three/drei three-stdlib

# 启动开发
npm run dev
```
