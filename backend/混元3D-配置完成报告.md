## 🔧 技术细节

### API接口

| 功能 | Action名称 | 说明 |
|------|-----------|------|
| 提交任务 | SubmitHunyuanTo3DRapidJob | 标准版（快速） |
| 提交任务 | SubmitHunyuanTo3DProJob | 专业版（高质量） |
| 查询任务 | QueryHunyuanTo3DRapidJob | 查询标准版任务 |
| 查询任务 | QueryHunyuanTo3DProJob | 查询专业版任务 |

### 请求参数

#### 图生3D
```python
req = models.SubmitHunyuanTo3DRapidJobRequest()
req.ImageBase64 = "base64编码的图片数据"
# 不要设置 req.Prompt
```

#### 文生3D
```python
req = models.SubmitHunyuanTo3DRapidJobRequest()
req.Prompt = "一只可爱的小狗"
# 不要设置 req.ImageBase64
```

### 响应格式

```json
{
  "Response": {
    "JobId": "1438492635039277056",
    "RequestId": "b0790e87-2871-440e-9ce1-55412d366340"
  }
}
```

### 任务状态

- `SUCCESS`: 任务成功，可下载GLB文件
- `FAIL`: 任务失败，查看ErrorMessage
- `PROCESSING`: 处理中，继续等待
- `PENDING`: 排队中，继续等待

## ❓ 常见问题

### Q1: 如何切换标准版和专业版？

A: 修改 `.env` 文件中的 `HUNYUAN3D_API_VERSION`：
- `rapid`: 标准版（快速）
- `pro`: 专业版（高质量）

### Q2: 任务失败怎么办？

A: 
1. 检查图片格式是否正确（jpg/png等）
2. 检查图片大小是否合适（建议不超过10MB）
3. 查看ErrorMessage了解具体原因
4. 联系腾讯云技术支持

### Q3: 如何查看资源包使用情况？

A: 访问：https://console.cloud.tencent.com/ai3d/resource-pack

### Q4: 生成的GLB文件在哪里？

A: 系统会自动下载到配置的输出目录，通常在 `backend/uploads/generation/`
