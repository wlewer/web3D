# 腾讯混元3D API权限配置指南

## 🔴 当前问题

测试返回错误：
```
AuthFailure.UnauthorizedOperation: 操作未授权，请检查CAM策略。
you are not authorized to perform operation (ai3d:SubmitHunyuanTo3DRapidJob)
```

**原因**：API密钥（SecretId/SecretKey）没有混元3D的操作权限。

---

## ✅ 解决方案：配置CAM权限

### 方法一：使用主账号（推荐用于测试）

如果您使用的是**主账号**的SecretId/SecretKey，主账号默认拥有所有权限，不需要额外配置。

**检查方法**：
1. 访问：https://console.cloud.tencent.com/cam/capi
2. 查看密钥类型
3. 如果显示"主账号密钥"，则无需配置权限
4. 如果显示"子用户密钥"，需要按方法二配置

---

### 方法二：为子用户配置权限

如果使用的是**子用户**的密钥，需要授予混元3D权限：

#### 步骤1：访问CAM控制台

打开浏览器，访问：https://console.cloud.tencent.com/cam

#### 步骤2：找到对应的子用户

1. 点击左侧菜单【用户】
2. 找到创建该密钥的子用户
3. 点击用户名进入详情

#### 步骤3：添加权限策略

1. 点击【关联策略】标签
2. 点击【添加策略】按钮
3. 搜索并选择以下策略之一：

   **选项A：完整权限（推荐）**
   - 策略名称：`QcloudAI3DFullAccess`
   - 说明：混元3D完全访问权限

   **选项B：只读权限**
   - 策略名称：`QcloudAI3DReadOnlyAccess`
   - 说明：只能查询，不能提交任务

4. 点击【确定】完成授权

#### 步骤4：等待生效

权限配置后通常需要**1-5分钟**生效。

---

### 方法三：使用自定义策略

如果需要更精细的权限控制，可以创建自定义策略：

#### 步骤1：创建自定义策略

1. 访问：https://console.cloud.tencent.com/cam/policy
2. 点击【新建自定义策略】
3. 选择【按策略语法创建】
4. 输入以下策略内容：

```json
{
    "version": "2.0",
    "statement": [
        {
            "effect": "allow",
            "action": [
                "ai3d:SubmitHunyuanTo3DRapidJob",
                "ai3d:SubmitHunyuanTo3DProJob",
                "ai3d:QueryHunyuanTo3DRapidJob",
                "ai3d:QueryHunyuanTo3DProJob"
            ],
            "resource": "*"
        }
    ]
}
```

5. 点击【下一步】
6. 填写策略名称：`Hunyuan3DAccess`
7. 点击【完成】

#### 步骤2：关联策略到用户

1. 回到用户列表
2. 找到对应的子用户
3. 点击【关联策略】
4. 选择刚创建的自定义策略
5. 点击【确定】

## ❓ 常见问题

### Q1: 如何确认是主账号还是子用户？

A: 访问 https://console.cloud.tencent.com/cam/capi，查看密钥列表中的"类型"列。

### Q2: 配置权限后仍然报错怎么办？

A: 
1. 等待5-10分钟让权限完全生效
2. 清除浏览器缓存后重新登录
3. 检查是否选择了正确的策略
4. 联系腾讯云技术支持

### Q3: 可以使用Root账号的密钥吗？

A: 可以，但不推荐。出于安全考虑，建议使用子用户并授予最小必要权限。

### Q4: 权限配置会影响已有的服务吗？

A: 不会。新配置的权限只会影响使用该密钥的新请求，不会影响正在运行的服务。
