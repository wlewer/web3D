# web3D

#### Description
随着3D Gaussian Splatting（3DGS）技术的快速发展，Web端3D渲染能力已达到前所未有的高度。World Labs于2026年4月开源的Spark 2.0，首次实现了在普通浏览器中流畅渲染亿级3DGS点云的能力。本方案旨在构建一个基于最新开源技术的3DGS技术官网，展示3DGS技术的强大能力，并提供手机拍照直接建模的创新功能。
